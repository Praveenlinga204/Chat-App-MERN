npm install express mongoose dotenv bycryptjs jsonwebtoken cors socket.io nodemon


Express - framework for building web servers and API's in node.js,.
Mongoose - library to interact with MongoDB database easily.
dotenv - loads environement variables for .env file.
bcryptjs -  bycrypt to hash password securely.
jsonwebtoken - creates authentication token. used for login systems and protected routes.
cors - allows frontend and backend to communicate from different domines.
socket.io - Enables real time communication between clint and server.
